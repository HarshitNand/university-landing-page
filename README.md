## 🚀 About the Project

This is a **modern University Landing Page** built using **React + Vite + Material UI**.  
It includes multiple sections like:

- Hero Section with Admission Form
- Stories & Highlights
- Latest Updates Timeline
- Why Choose Us
- Research Section
- Testimonials
- Chancellor’s Desk
- News Section

---

## 🛠 Built With

This project was built using:

- ⚛️ React (Vite)
- 🎨 Material UI (MUI)
- 💡 JavaScript (ES6)
- 🎯 CSS (Custom + MUI SX)
- 🧑‍💻 VS Code
